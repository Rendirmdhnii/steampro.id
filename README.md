# SteamPro.id - Halaman Penawaran

Proyek ini adalah antarmuka web responsif untuk SteamPro.id, sebuah pusat penyedia paket usaha setrika uap dan binatu. Halaman ini dirancang untuk menampilkan katalog paket usaha, simulasi perhitungan omzet, dan ulasan pelanggan dengan pendekatan desain yang mengutamakan perangkat seluler.

## Fitur Utama

- **Tata Letak Responsif:** Tampilan yang menyesuaikan secara otomatis untuk perangkat seluler, tablet, dan komputer meja.
- **Katalog Paket:** Menampilkan pilihan paket usaha beserta harga dan daftar fasilitas lengkap.
- **Simulasi Keuangan:** Tabel proyeksi laba bersih bulanan untuk usaha setrika uap dan binatu.
- **Ulasan Pelanggan:** Kartu ulasan dari klien yang telah menggunakan layanan.

## Teknologi yang Digunakan

- Next.js
- TypeScript
- Tailwind CSS
- Lucide React (untuk ikon pelengkap)
